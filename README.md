# toki-rabbit
> RabbitMQ plugin used in Toki core.

This plugin interfaces Toki core with the [BunnyBus](https://github.com/xogroup/bunnybus) RabbitMQ driver.

Lead Maintainer: [Wes Tyler](https://github.com/westyler)
